2020-01-10

Go 中不常注意的各种细节集锦

打印true还是false?

package main

import "fmt"

func f() bool {
	return false
}
func main() {
	switch f()
	{
	case true:
		fmt.Println(true)
	case false:
		fmt.Println(false)
	}
}

Go语言分号自动插入规则

1.在 Go 代码中，注释除外，如果一个代码行的最后一个语法词段（token）为下列所示之一，则一个分号将自动插入在此字段后（即行尾）：
一个标识符
一个整数、浮点数、虚部、码点或者字符串字面表示形式
这几个跳转关键字之一：break、continue、fallthrough 和 return
自增运算符++或者自减运算符–
一个右括号：) ] }
2.为了允许一条复杂语句完全显示在一个代码行中，分号可能被插入在一个右小括号)或者右大括号}之前
